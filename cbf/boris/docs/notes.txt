Runtime configuration file is project/boris/conf/config.php
Files generated by the Propel build process - project-classmap.php & project-conf.php
- must go to project/boris/conf/ 
Class files generated by Propel build must go to project/boris/classes/

boris/system contains third party libraries required by the boris framework
boris/system/propel - http://www.propelorm.org/
boris/system/HTML/Template/ contains the Flexy templating system


Required Libraries:
phing
Can also just install propel from distro package system
    copy /usr/share/php5/propel/ to project/boris/system/propel/


Building:
In boris/build:
Make a directory for your build situation
Configure build.properties
